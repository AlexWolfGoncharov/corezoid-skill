# Corezoid Templates

Reusable process skeletons for fast project bootstrap.

## Purpose

- Start from a known-safe structure instead of empty canvas.
- Keep consistent node lanes (main/retry/error).
- Speed up delivery while preserving readability and standards.

## Suggested Usage

1. Pick a template closest to the target use-case.
2. Rename nodes with domain-specific titles/descriptions.
3. Fill in logic details and API integrations.
4. Run final checks from `SKILL.md` validation checklist.
