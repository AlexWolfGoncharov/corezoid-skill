# Corezoid JSON Schema

Schema-based validation guidance for process JSON structures.

## Purpose

- Catch structural errors before deployment.
- Validate node references and required fields.
- Enforce consistent JSON shape for process import/export workflows.

## Suggested Usage

1. Build or adapt process JSON.
2. Validate required fields and node graph consistency.
3. Resolve schema violations before final handoff.
